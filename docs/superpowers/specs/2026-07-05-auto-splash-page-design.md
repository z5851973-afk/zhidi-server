# 自动开屏页设计

## 目标

将现有需要点击按钮才能进入首页的欢迎页，改为具有品牌展示作用的自动开屏页。页面参考确认稿的暖色居家氛围，启动后无需用户操作。

## 视觉结构

- 背景使用现有 `assets/splash_bg.png` 全屏裁切，并叠加浅暖色遮罩，保证文字在明暗区域都清晰。
- 品牌内容整体置于屏幕视觉中心略偏上位置。
- 从上到下依次展示 Logo、“知底”、主标语“装修找知底，心里就有底”和价值说明“工人透明｜工价透明｜工艺透明｜平台保障”。
- 保留橙色作为 Logo 和重点文字色，其余文字使用深灰与中灰。
- 删除“开启安心装修之旅”按钮和底部“平台托底”文案，避免自动跳转页面出现可操作暗示。

## 时序与交互

- 页面显示满 2 秒后开始退出。
- 使用 250ms 透明度淡出，再切换到首页。
- 页面不响应点击跳过，避免误触导致品牌内容闪现。
- 计时器在页面销毁时取消，不在已卸载页面上触发回调。
- 每次应用冷启动显示；本轮不增加“首次启动才显示”的持久化规则。

## 适配

- 背景使用 `BoxFit.cover`，允许边缘裁切但不可拉伸。
- 内容使用安全区和相对间距，在窄屏、小高度及大字体环境下不溢出。
- Logo 使用现有 `assets/logo.png`，不另生成近似 Logo。
- 状态栏保持透明，图标使用深色以适配浅暖背景。

## 验收标准

- 启动后无需点击，2 秒后自动开始淡出并进入首页。
- 淡出时长为 250ms，回调只触发一次。
- 页面不显示按钮或其他可点击控件。
- 320px 和 390px 宽度下无布局溢出。
- Logo、品牌名、主标语和四项价值说明均可见。
